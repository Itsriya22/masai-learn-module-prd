# Masai Learn Module-wise Lecture & Session PRD

This repository contains a Product Requirement Document for a Masai Learn feature that groups lectures and sessions module-wise, displays module progress, and surfaces upcoming session alerts.

## What’s included

- `MasaiLearn_Module_PRD.md` — full PRD describing objectives, user stories, functional and non-functional requirements, UX behavior, metrics, dependencies, and implementation considerations.

## Feature Summary

The feature enables students to:

- view course content organized by module
- see module-level completion progress
- access lecture and live session details in one place
- receive alerts for upcoming scheduled sessions

## Why this matters

This improves course structure visibility, helps learners plan study sessions, reduces missed live events, and boosts progress tracking across modules.
